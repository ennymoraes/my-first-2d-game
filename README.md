Dodge the Creeps!

My first 2D game built with Godot Engine, following the official beginner tutorial :)
The player must dodge enemies that spawn from the edges of the screen for as long as possible. The longer you survive, the higher your score!

Built with:
Godot Engine 4.6

What I learned:

- Player movement and physics
- Spawning enemies with timers
- Signals and scene instancing
- HUD with score and messages
- Game over and restart logic
